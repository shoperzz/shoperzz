# Tooling — Shared Configurations & Scripts

This directory contains development tools and shared configurations for the Shoperzz monorepo.

## Contents
- **Base Configurations**: TypeScript (`tsconfig.base.json`), ESLint, Prettier, Jest.
- **Maintenance Scripts**: Scripts for global builds, release management, and security audits.

## Role
It acts as the technical control tower. By centralizing configurations here, we ensure every package in the monorepo follows the same quality standards without duplication.
