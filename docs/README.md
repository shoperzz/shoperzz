# Documentation (@shoperzz/docs)

This directory contains all the Markdown source files for the Shoperzz project documentation.

## Purpose

- **Centralized Knowledge**: Serves as the single source of truth for how Shoperzz works, how to use it, and how to contribute.
- **Developer Guides**: Provides comprehensive guides for setting up development environments, understanding core concepts, and building plugins.
- **API References**: Houses detailed API documentation for various Shoperzz components and plugins.
- **Community Resources**: Includes guidelines for contribution, security policies, and code of conduct.

## Structure

The documentation is organized into logical sections to facilitate navigation and understanding. Each subdirectory within `docs/` typically represents a major section of the documentation (e.g., `guides/`, `api-reference/`, `plugins/`).

## Contribution

Contributions to the documentation are highly encouraged! Clear and comprehensive documentation is vital for the success of any open-source project. Please refer to the main `CONTRIBUTING.md` for general contribution guidelines.
