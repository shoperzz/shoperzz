# Orchestrator CLI (@shoperzz/create)

The CLI project represents the gateway to Technical Sovereignty for our engineers.

Built with the Clack.js interface, this interactive tool handles the initialization and scaffolding of the e-commerce project at the end-developer's machine. Ultimately, `npx @shoperzz/create` will provision automatic workspaces, configure Turborepo, install requested extensions (plugin packages), and bootstrap the environment.
